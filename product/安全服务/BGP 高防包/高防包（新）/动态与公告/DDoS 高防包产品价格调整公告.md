腾讯云 DDoS 高防包价格策略调整，新策略将于**2022年03月08日**正式启用。

## 调整内容
- 计费模式调整
- 售卖周期调整
- 防护 IP 数规格调整
- 刊例价调整

## 内容详情
### 计费模式调整
调整后产品按“IP 数量+业务规模”计费。

| 调整前/后 | 计费模式                  |
| --------- | ------------------------- |
| 调整前    | 防护 IP 数                |
| 调整后    | 防护 IP 数 + **业务规模** |


#### 业务规模计费模式说明
业务规模可选规格：50Mbps、100 Mbps、200Mbps、300 Mbps、500 Mbps、600Mbps、800 Mbps、1 Gbps、2 Gbps、3 Gbps。

#### 售卖周期调整
调整后产品支持按年起售。

| 调整前/后| 防护有效期（可选规格） |
| ------------- | --------------------------- |
| 调整前        | 3个月、6个月、1年、2年、3年 |
| 调整后        | 1年、2年、3年               |
